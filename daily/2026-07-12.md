### 2. 网页端路径解析与排错实战 🔍
*   **时序图引入排错**：在 `scope.md` 中引入实操波形截图时，遇到了“本地预览正常但网页端破裂”的经典 404 Bug。
*   **核心原因与收获**：
    1.  **路径分隔符**：Windows 系统下的反斜杠 `\` 在 Linux Web 服务器上无法被识别，必须统一写成标准的正斜杠 `/`。
    2.  **多级嵌套路径**：因为笔记被挪进了 `tools/toolchains/`（两层深度），原本退回一级的 `../images/` 只能退到 `tools/`，找不到图片。必须使用二级相对路径 **`../../images/`** 才能回溯到项目根目录。
    *   *注：修正后本地预览与 Docsify 网页端均已完美显示，且 Python 压缩脚本在自动替换文件名时完美兼容。*

### 3. 一体化集成运维中枢建设 (DevOps Integration) 🚀
*   **工具链大合并**：将“新建日志”与“图片智能压缩替换”两个独立的 Python 脚本合并为统一的 **`manage.py`** 命令行中枢，根目录文件由 4 个瘦身至 2 个。
*   **一键开发沙盒（超级加分项）**：自动生成 `.vscode/tasks.json` 并配置 `"runOn": "folderOpen"`。首次实现：双击 `.bat` 后打开 VS Code 的同时，**VS Code 内部集成终端（Terminal）自动静默拉起网页服务器**，不再弹窗外部多余的黑窗口，实现了完全无感的纯净 IDE 体验。
*   **一键打包部署**：集成了 Git 一键 Stage、带有自动时间戳的 Commit 提交、以及一键推送到远程 GitHub 的完整闭环。
