# C 语言 - 核心运算符专题

本专题聚焦于 C/C++ 开发中最基础且高频使用的四类核心运算符：**内存大小运算符（`sizeof`）**、**条件运算符（`?:`）**、**指针运算符（`&` 与 `*`）**，以及**结构体成员访问运算符（`.` 与 `->`）**。通过对比 C 与 C++ 的代码实现，帮助您深入理解这些运算符的底层逻辑与应用场景。

---

## 一、 核心运算符解析

### 1. 内存大小运算符：`sizeof`
*   **本质**：`sizeof` 是一个**单目运算符**，而非函数。它在编译时计算出对象或类型所占用的字节数。
*   **格式占位符**：
    *   在 C 语言中，`sizeof` 返回值的类型是 `size_t`（无符号整型），在 `printf` 中标准推荐使用 **`%zu`** 作为占位符。
    *   在 C++ 中，直接通过 `std::cout` 即可自动识别并输出。

### 2. 条件运算符：`?:`
*   **本质**：C 语言中唯一的**三目运算符**（需要三个操作数）。
*   **语法**：`表达式1 ? 表达式2 : 表达式3`
*   **逻辑**：若 `表达式1` 为真，则整个表达式的值为 `表达式2` 的值；否则为 `表达式3` 的值。常用于替代简单的 `if-else` 语句以简化代码。

### 3. 指针运算符：`&` 与 `*`
*   **取地址运算符 `&`**：用于获取变量在内存中的物理起始地址。在 `printf` 中，输出地址需要使用 **`%p`** 占位符，并将指针转换为 `(void*)` 以确保兼容性。
*   **解引用运算符 `*`**：作用于指针变量，用于访问或修改该指针所指向的内存空间中存储的数据。

### 4. 成员访问运算符：`.` 与 `->`
这两个运算符都用于访问结构体（或类）的成员，但使用对象不同：
*   **点运算符 `.`（直接访问）**：适用于**结构体普通变量**。
*   **箭头运算符 `->`（间接访问）**：适用于**指向结构体的指针变量**。
*   **等价关系**：`ptr->member` 在物理和逻辑上完全等价于 `(*ptr).member`（先解引用获取结构体变量，再用点运算符访问其成员）。

## 二、 C 语言代码实现

在 C 语言中，我们需要注意原生标准库的占位符使用，以及通过 `typedef` 来简化结构体的定义。

```c
#include <stdio.h>

// 在 C 语言中，常用 typedef 来定义结构体，这样在声明变量时就可以省略 struct 关键字
typedef struct {
    int id;
    int age;
} Student;

int main() {
    // ==========================================
    // 1. sizeof 运算符应用
    // ==========================================
    printf("--- sizeof 应用 ---\n");
    int a = 10;
    
    // %zu 是 C 语言中标准推荐的用于输出 sizeof 返回值（size_t 类型）的占位符
    printf("int 类型占用的字节数: %zu 字节\n", sizeof(int));
    printf("变量 a 占用的字节数: %zu 字节\n", sizeof(a));
    printf("Student 结构体占用的字节数: %zu 字节\n", sizeof(Student));

    // ==========================================
    // 2. ?: 三目运算符应用
    // ==========================================
    printf("\n--- ?: 三目运算符应用 ---\n");
    int x = 15;
    int y = 20;
    // 如果 x > y，则 max_val 等于 x，否则等于 y
    int max_val = (x > y) ? x : y;
    printf("较量后的较大值是: %d\n", max_val);

    // ==========================================
    // 3. & (取地址) 与 * (解引用) 应用
    // ==========================================
    printf("\n--- & 与 * 应用 ---\n");
    int number = 99;

    // 使用 & 获取 number 的内存地址，并存入指针变量 ptr 中
    int *ptr = &number;

    printf("number 的当前数值: %d\n", number);
    // %p 用于输出指针的地址值（通常以十六进制显示）
    // 在标准 C 中，打印指针时最好将指针强制类型转换为 (void*) 以提高兼容性
    printf("number 在内存中的地址 (&number): %p\n", (void*)ptr);

    // 使用 * 解引用指针，直接修改该内存地址里存储的数据
    *ptr = 200;
    printf("修改后 number 的实际数值: %d\n", number);

    // ==========================================
    // 4. . 与 -> 成员访问运算符应用
    // ==========================================
    printf("\n--- . 与 -> 应用 ---\n");

    // 声明一个结构体变量 stu1
    Student stu1;
    stu1.id = 1001;  // 使用 '.' 直接访问并赋值
    stu1.age = 18;   // 使用 '.' 直接访问并赋值
    printf("普通结构体变量 stu1 的初始年龄: %d\n", stu1.age);

    // 创建一个指向 stu1 的结构体指针 stu_ptr
    Student *stu_ptr = &stu1;

    // 使用 '->' 通过指针间接修改成员变量的值
    stu_ptr->age = 19;

    // 上述代码与下行代码是完全等价的：
    // (*stu_ptr).age = 19;

    printf("通过指针修改后，stu1 的年龄变为: %d\n", stu1.age);

    return 0;
}
```

## 三、 C++ 语言代码实现

在 C++ 中，我们利用 `std::cout` 的类型自动识别特性来观察这些运算符的输出。

```cpp
#include <iostream>

// 定义一个简单的结构体（表示学生信息）
struct Student {
    int id;
    int age;
};

int main() {
    // ==========================================
    // 1. sizeof 运算符应用
    // ==========================================
    std::cout << "--- sizeof 应用 ---" << std::endl;
    int a = 10;
    std::cout << "int 类型占用的字节数: " << sizeof(int) << " 字节" << std::endl; // 通常输出 4
    std::cout << "变量 a 占用的字节数: " << sizeof(a) << " 字节" << std::endl;     // 输出 4
    std::cout << "Student 结构体占用的字节数: " << sizeof(Student) << " 字节" << std::endl; // 通常输出 8 (4 + 4)

    // ==========================================
    // 2. ?: 三目运算符应用
    // ==========================================
    std::cout << "\n--- ?: 三目运算符应用 ---" << std::endl;
    int x = 15;
    int y = 20;
    // 如果 x > y，则 max_val 等于 x，否则等于 y
    int max_val = (x > y) ? x : y; 
    std::cout << "较大值是: " << max_val << std::endl; // 输出 20

    // ==========================================
    // 3. & (取地址) 与 * (解引用) 应用
    // ==========================================
    std::cout << "\n--- & 与 * 应用 ---" << std::endl;
    int number = 99;
    
    // 使用 & 获取 number 的内存地址，并存入指针变量 ptr
    int* ptr = &number; 
    
    std::cout << "number 的数值: " << number << std::endl; // 输出 99
    std::cout << "number 在内存中的地址 (&number): " << ptr << std::endl; // 输出如 0x7ffeefbff56c 的十六进制地址
    
    // 使用 * 解引用指针，修改该地址里的数值
    *ptr = 200; 
    std::cout << "修改后 number 的实际数值: " << number << std::endl; // 输出 200

    // ==========================================
    // 4. . 与 -> 成员访问运算符应用
    // ==========================================
    std::cout << "\n--- . 与 -> 应用 ---" << std::endl;
    
    // 实例化一个结构体普通对象
    Student stu1;
    stu1.id = 1001;  // 使用 '.' 直接访问成员变量
    stu1.age = 18;   // 使用 '.' 直接访问成员变量
    std::cout << "普通对象 stu1 的年龄: " << stu1.age << std::endl;

    // 创建一个指向 stu1 的指针
    Student* stu_ptr = &stu1;
    
    // 使用 '->' 通过指针间接修改成员变量的值
    stu_ptr->age = 19; 
    
    // 以下写法与 stu_ptr->age = 19 完全等价：
    // (*stu_ptr).age = 19; 

    std::cout << "通过指针修改后，stu1 的年龄变为: " << stu1.age << std::endl; // 输出 19

    return 0;
}
```

## 四、 核心知识点对比与总结

| 运算符 | 物理意义 | 典型操作对象 | 常见格式化/输出细节 |
| :--- | :--- | :--- | :--- |
| **`sizeof`** | 获取数据类型或变量所占内存大小 | 类型名、变量名、结构体名 | C 语言中推荐使用 `%zu` 打印其返回值 |
| **`?:`** | 基于条件判断进行分支选择 | 关系/逻辑表达式、两个待选值 | 适合简化简单的 `if-else` 返回赋值语句 |
| **`&`** | 获取变量的首地址（门牌号） | 任意类型的非寄存器普通变量 | C 语言中使用 `%p` 输出，参数需强转为 `(void*)` |
| **`*`** | 顺着物理地址访问或改写其内存数据 | 指针变量 | 执行解引用前，必须确保指针不是 `NULL` |
| **`.`** | 直接访问结构体/类的内部成员 | 结构体普通变量（实例） | 优先级极高，结合解引用时需加括号，如 `(*ptr).member` |
| **`->`** | 间接（通过指针）访问结构体内部成员 | 指针变量 | 底层相当于先执行解引用，再进行成员访问 |
