## 🙋‍♂️ 面试高频问题

### Q1: 什么是变量的地址？指针变量存的是什么？
*   **回答**：
    *   在内存中，每个存储单元都有一个唯一的编号，这个编号就是**地址**。
    *   **指针变量**是一种特殊的变量，它里面存储的不是普通数值（如整数、字符），而是**另一个变量的内存地址**。

---

## 💻 核心代码测试

以下是今天测试的经典代码：指针的声明、赋值与取值。

```c
#include <stdio.h>

int main() {
    int a = 10;      // 声明一个普通整型变量
    int *p = NULL;   // 声明一个整型指针变量，初始化为 NULL

    p = &a;          // 将变量 a 的地址赋给指针 p

    printf("变量 a 的值: %d\n", a);
    printf("变量 a 的地址: %p\n", &a);
    printf("指针 p 存储的地址: %p\n", p);
    printf("通过指针 p 访问 a 的值 (*p): %d\n", *p);

    return 0;
}
