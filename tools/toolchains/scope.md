这里为您整理了一份**示波器（Oscilloscope）基础操作与入门指南**。

示波器是电子工程师最常用的仪器之一，与逻辑分析仪不同的是，它不仅能看数字信号的“高低电平”，还能精确测量**模拟信号的电压变化细节**（如噪声、过冲、波形畸变等）。

# 示波器（Oscilloscope）基础操作与入门指南

本指南梳理了示波器的核心控制区域、常用按键功能以及标准的操作流程，旨在帮助建立对示波器的初步认知。

## 一、 示波器与逻辑分析仪的区别

| 特性 | 逻辑分析仪 (Logic Analyzer) | 示波器 (Oscilloscope) |
| :--- | :--- | :--- |
| **测量对象** | 仅数字信号（0 和 1） | 模拟信号 + 数字信号 |
| **关注点** | 协议分析、多通道时序关系 | 电压幅值、波形畸变、噪声、频率细节 |
| **通道数** | 通常较多（8通道、16通道或更多） | 通常较少（2通道或4通道） |
| **电压细节**| 无法显示具体电压数值（只分高低） | 能精确测出实时电压（如 3.12V、3.3V、5V） |

---

## 二、 控制面板三大核心区域

示波器的面板看起来复杂，但 90% 的常用操作都集中在以下三个控制区（通常以不同的颜色标识区分）：

```
+-------------------------------------------------------------+
|                                                             |
|   +-----------------------+      [Vertical]    [Horizontal] |
|   |                       |       ( Volts )     (  Time  )  |
|   |                       |        (Pos)         (Pos)      |
|   |        SCREEN         |                                 |
|   |                       |      [Trigger]     [Utility]    |
|   |                       |       (Level)     [Auto][Single]|
|   +-----------------------+                                 |
+-------------------------------------------------------------+
```

### 1. 垂直控制区（Vertical）—— 调整波形的“高矮”
* **`VOLTS/DIV` 旋钮（电压分辨率）**：
  * **作用**：调整屏幕上垂直方向每格代表的电压值。
  * **操作**：顺时针旋转波形变高（分辨率变小），逆时针旋转波形变矮（分辨率变大）。
* **`POSITION` 旋钮（垂直位移）**：
  * **作用**：控制波形在屏幕上的基准线上下移动。
* **`CH1 / CH2 / CH3 / CH4` 按键**：
  * **作用**：打开或关闭对应的物理通道。

### 2. 水平控制区（Horizontal）—— 调整波形的“宽窄”
* **`SEC/DIV` 或 `TIME/DIV` 旋钮（时基 / 时间分辨率）**：
  * **作用**：调整屏幕上水平方向每格代表的时间值。
  * **操作**：顺时针旋转波形在水平方向拉伸（看得更细），逆时针旋转波形在水平方向压缩。
* **`POSITION` 旋钮（水平位移）**：
  * **作用**：控制波形左右平移，以便观察不同时间段的信号细节。

### 3. 触发控制区（Trigger）—— 让波形“稳定”不闪烁
* **`LEVEL` 旋钮（触发电平）**：
  * **作用**：调整触发阈值。在屏幕上表现为一根可以上下移动的虚线。只有当波形的电压穿过这根线时，示波器才会开始捕获并显示稳定的波形。
* **`50%` 或 `Force Trigger` 按键**：
  * **作用**：自动将触发电平设置在波形正中间，快速稳定波形。

## 三、 核心按键与特色功能

### 1. `AUTO` (自动设置)
* **作用**：示波器的“一键智能自动配置”。
* **适用场景**：当您接入信号后，屏幕上只有一条杂乱的线或波形超出屏幕时，按下 `AUTO`，示波器会自动调整垂直分度、时基和触发，将波形调整到屏幕中央。
* *提示*：对于已知信号，手动调整能学到更多；遇到未知信号时，可使用 `AUTO` 快速定位。

### 2. `SINGLE` (单次触发 / 捕获单次事件)
* **作用**：让示波器进入“等待触发并只捕获一次”的状态。
* **工作逻辑**：
  1. 按下 `Single`，示波器处于等待状态（此时屏幕上通常显示亮红色的 **`Ready`**）。
  2. 信号线上发生了一次满足触发条件的电平跳变（例如系统开机的瞬间上电冲击、或者按键落下的瞬间电平）。
  3. 示波器捕获这**唯一一次**的完整过程，并将波形冻结在屏幕上（显示 **`Stop`**），不再继续采集。
* **适用场景**：捕获单次脉冲、开机瞬间电压尖峰（Surge）、非周期性的突发故障信号等。

### 3. `RUN/STOP` (运行/停止)
* **作用**：暂停或恢复屏幕上波形的实时刷新。
* **区别**：`Stop` 是保留当前屏幕上的最后一帧图像；而 `Single` 是等待下一次符合条件的信号来敲门时才进行单次捕捉。

### 4. `MEASURE` (测量菜单)
* **作用**：调出自动测量选项。
* **功能**：可以直接在屏幕边缘实时显示波形的 **峰峰值（Vpp）**、**最大电压（Vmax）**、**频率（Freq）**、**周期（Period）**、**占空比**等数值，无需用格数手动计算。

## 四、 示波器基础使用操作流程

一个规范的示波器操作通常遵循以下五步法：

```
+----------+      +----------+      +----------+      +----------+      +----------+
| 1.自检校准| ---> | 2.接入信号| ---> | 3.一键自动| ---> | 4.手动微调| ---> | 5.单次捕获|
| (探头1KHz) |      | (信号+GND) |      |  (AUTO)  |      | (V/H旋钮) |      | (Single) |
+----------+      +----------+      +----------+      +----------+      +----------+
```

### 步骤 1：探头补偿与自检（使用前确认）
1. 将示波器探头连接到通道 1（`CH1`）。
2. 将探头的钩子挂在示波器自带的 **`1KHz / 3V` 校准信号输出夹**（通常在面板右下角）上，探头的接地夹夹在旁边的金属接地片上。
3. 按下 **`AUTO`**。
4. 如果屏幕上显示出边缘非常平整、规整的 1KHz 方波，说明探头和示波器工作正常，可以开始测量。

### 步骤 2：接入待测信号
1. **接地（共地）**：将探头的黑色接地夹子连接到被测电路板的 **`GND`** 上。
2. **连接测试点**：将探头前端的钩子/针尖接触要测量的信号源引脚。

### 步骤 3：获取初始波形
* 按下 **`AUTO`** 键，让示波器自动抓取并呈现波形。

### 步骤 4：调整至最佳视觉效果
1. 观察波形高度：调整 **`VOLTS/DIV`** 旋钮，使波形占屏幕高度的 1/2 到 2/3 左右。
2. 观察波形密度：调整 **`SEC/DIV`** 旋钮，使屏幕上显示 2 到 5 个完整的波形周期。
3. 调整波形位置：利用垂直和水平的 **`POSITION`** 旋钮，将要观察的波形细节移到屏幕正中央。

### 步骤 5：使用 Single 抓取突发信号（如：电源开机瞬间电压）
1. 连好接线（探头接在电源输出端，地接电源地）。
2. 手动调节 **`LEVEL`** 旋钮，将触发电平虚线移到略低于期望开机电压的位置（例如，期望测 5V 开机，电平设在 2.5V 左右）。
3. 按下 **`SINGLE`** 按键，示波器进入等待状态。
4. 开启被测电路板的电源开关。
5. 示波器自动触发并锁定开机瞬间的电压波形，此时可以放大、缩小波形，观察上电是否存在电压过冲或不稳定的振荡。
